# TaşıTır - Lojistik ve Taşımacılık Platformu (Frontend)

TaşıTır, lojistik ve taşımacılık süreçlerini dijitalleştirerek tedarik zinciri yönetimini kolaylaştırmayı amaçlayan web ve mobil tabanlı bir platformdur. Bu depo (repository), projenin kullanıcı arayüzü (frontend) kaynak kodlarını içermektedir.

## Özellikler

* **Dinamik Yönetim Paneli:** Nakliye talepleri, araç durumları ve sevkiyat rotalarının takibi için optimize edilmiş arayüzler.
* **Gelişmiş Filtreleme:** İlanlar, yük tipleri ve taşımacılık rotaları için özelleştirilmiş arama motoru bileşenleri.
* **Duyarlı (Responsive) Tasarım:** Mobil, tablet ve masaüstü ekran boyutlarıyla tam uyumlu, kullanıcı deneyimi odaklı tasarım.
* **Modüler Bileşen Mimarisi:** Yeniden kullanılabilir, performanslı ve genişletilebilir UI bileşenleri.

## Teknolojik Altyapı

* **Kütüphane/Framework:** React / React Native
* **Programlama Dilleri:** TypeScript / JavaScript / HTML5 / CSS3
* **Durum Yönetimi & Haberleşme:** Context API / Redux & Axios

## Kurulum ve Çalıştırma

Projenin yerel ortamda sorunsuz çalışabilmesi için sisteminizde **Node.js** yüklü olmalıdır.

1. **Depoyu Klonlayın:**
   ```bash
   git clone https://github.com/fatihmbnn/TasiTir.git
   cd TasiTir
   ```

2. **Bağımlılıkları Yükleyin:**
   ```bash
   npm install
   # veya
   yarn install
   ```

3. **Uygulamayı Başlatın:**
   ```bash
   npm start
   # veya
   yarn start
   ```

> **Not:** Bu proje geliştirme süreçlerini tamamlamış olup, şu anda aktif olarak güncellenmemektedir. Geçmiş bir çalışma ve portfolyo ögesi olarak arşivlenmiştir.
